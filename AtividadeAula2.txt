EXERCICIO

Com base no exemplo desenvolvido lab para o banco de dados PLANTS. Crie:

Uma estrutura lógida de Vanco de Daods para uma Imobiliária armazenar os registros de seus imóveis.
Para isto crie uma tabela IMOVEL com as seguintes colunas:
- Id do tipo inteiro
- Nome do tipo string
- Descricao do tipo string
- Nro de Quartos do tipo inteiro
- Valor de venda do tipo real ou float

Insira ao menos 10 registros de dados para esta entidade;

Aplique ao menos 3 tipos diferentes de consulta para a exibição dos dados.
